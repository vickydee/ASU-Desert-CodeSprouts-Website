# ASU Robotics Camps Website

This is a standalone Vite + React app for the ASU Robotics Camps site.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`

## Build Static Assets

1. Create a production build: `npm run build`
2. Deploy the generated `dist/` directory to any static web server.
3. Optional local preview: `npm run preview`
